# set the band names for a vrt_x object

set the band names for a vrt_x object

## Usage

``` r
vrt_set_band_names(x, band_names)
```

## Arguments

- x:

  A vrt_block, vrt_stack, or vrt_collection object

- band_names:

  A character vector of the band names

## Value

A modified object of the same class as `x` with band names set.
