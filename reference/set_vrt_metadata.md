# Internal function to set metadata in a VRT

Internal function to set metadata in a VRT

## Usage

``` r
set_vrt_metadata(x, keys, values, as_file = FALSE)
```

## Arguments

- x:

  A character string of the path to the VRT

- keys:

  A character vector of the metadata keys

- values:

  A character vector of the metadata values

- as_file:

  A logical indicating if the VRT should be saved to a file

## Value

An xml_document object
