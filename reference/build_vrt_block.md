# constructor for vrt_block class

constructor for vrt_block class

## Usage

``` r
build_vrt_block(x, maskfun = NULL, pixfun = NULL, warped = FALSE, ...)
```

## Arguments

- x:

  An xml_document object

- maskfun:

  A function of the mask function

- pixfun:

  A function of the pixel function

- warped:

  A logical indicating whether the VRT is warped

- ...:

  Additional arguments not used
