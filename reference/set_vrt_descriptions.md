# Internal function to set the descriptions for the bands in a VRT

Internal function to set the descriptions for the bands in a VRT

## Usage

``` r
set_vrt_descriptions(x, descriptions, as_file = FALSE)
```

## Arguments

- x:

  A character string of the path to the VRT

- descriptions:

  A character vector of the descriptions

- as_file:

  A logical indicating if the VRT should be saved to a file

## Value

A character string of the path to the modified VRT
